# SPLYZA 調査レポート

**作成日**: 2025年12月24日

## 1. 会社概要

SPLYZAはスポーツ向け動画分析・コミュニケーションツールを提供する日本の企業です。

### 採用実績
- **SPLYZA Teams**: 650+チーム導入
- **SPLYZA Motion**: 100+団体導入（※公式サイト記載）、約200団体（※検索結果）

## 2. 製品ラインナップ

### 2.1 SPLYZA Teams - スポーツ動画分析・コミュニケーションツール

チーム単位で利用する動画分析・共有ツールです。11の機能を提供します。

#### 11の機能一覧

| 機能 | 説明 |
|-----|------|
| **1. 描き込み** | スポーツ解説番組のように動画に書き込み可能 |
| **2. タグ付け** | 「見たいシーン」を素早く頭出し・検索可能 |
| **3. クイズ機能** | 選手の理解度を確認するためのクイズ作成 |
| **4. デバイス横断・同時編集** | 異なるデバイスから同時に編集可能 |
| **5. データ集計・分析** | タグ付けしたデータを集計・分析 |
| **6. ライブタグ** | 撮影しながらリアルタイムでタグ付け |
| **7. ダウンロード機能** | 動画のダウンロード |
| **8. 結合アップロード** | 複数の動画を結合してアップロード |
| **9. ルーム** | チーム内での共有・コミュニケーション機能 |
| **10. プレイリスト作成** | 動画をまとめたプレイリスト作成 |
| **11. AIまとめ機能** | AIによる自動要約機能 |

---

### ライブタグ機能（詳細）

**最大の特徴**: 撮影しながらリアルタイムでタグ付けが可能

- ハーフタイムでデータと映像の双方で分析できる
- 最小限の時間で試合の振り返りが可能
- タグスキップ機能でハイライトシーンだけを閲覧可能

**使用フロー（バスケットボールの例）**:
1. 試合中にiPadなどで撮影
2. オフェンス/ディフェンスなどのタグをリアルタイムで入力
3. ハーフタイムですぐにタグ付けしたシーンを振り返り可能

---

### タグ付け機能（詳細）

**主な機能**:
- 試合や練習のシーンにタグ付け
- タグスキップ機能で該当シーンのみ再生
- 振り返りたいシーンだけをピックアップ可能

**活用シーン**:
- 試合振り返り
- 選手個人の成長記録
- 戦術の分析・改善

## 2.2 SPLYZA Motion - AI 3Dモーション解析アプリ

iOS向けのマーカーレス3Dモーション解析アプリ。

### 特徴

- **単一カメラで3Dモーションキャプチャ可能**
  - 特別な機器不要
  - iPhone/iPadのみで実現

- **AIによる高度な解析**
  - 2D・3Dモーション分析
  - 角度測定

- **2024年9月に3D解析機能追加**

### 使用方法

1. **撮影**: 120fpsスローモーションで最大20秒間撮影（横向き）
2. **解析**: アプリ起動でAIが自動解析
3. **結果**: 詳細な2D・3Dモーション分析結果を取得

### 対応デバイス

- iOS（iPhone / iPad）
- App Store ID: 1550889336

### 活用目的

- 現場での迅速な動作解析
- スポーツ科学への容易なアクセス
- プロフェッショナルグレードのモーション解析をモバイルデバイスで

## 5. SPLYZAの課題（考察）

### 5.1 高価格帯

- 最安値でも¥16,500/月
- Video Noteの¥1,000/月と比較すると16.5倍の価格差
- 個人利用にはハードルが高い

### 5.2 チーム単位での契約が必要

- 個人での導入が想定されていない可能性
- 導入にはチーム内での合意形成が必要

### 5.3 学習コスト

- 11機能と多機能なため習得に時間がかかる可能性
- 運用前ミーティング・説明会が必須となっている

4. **チーム機能の価値**
   - 同時編集・ルーム機能はチーム利用で必須

## 8. まとめ

SPLYZAはスポーツ特化型の動画分析ツールとして、チーム利用・教育用途に強みを持っています。特にライブタグ・クイズ機能・同時編集など、チームでの利用に特化した機能が充実しています。

一方で、価格が¥16,500/月と高価格帯であり、個人利用にはハードルが高いという課題もあります。

Video Noteは、**「低価格で個人から利用できる」「シンプルで使いやすい」**という点で差別化が可能です。将来的にタグ付け機能や同時編集機能などの実装を検討することで、より競争力のある製品になるでしょう。
