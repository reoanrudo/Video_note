## 1. エグゼクティブサマリー

### 主要発見

1. **市場は2つのカテゴリーに明確に分割**
   - **スポーツ分析用ツール**: 高価格（$400-$50,000/年）、専門機能重視
   - **汎用コラボレーションツール**: 中価格（$15-25/月）、チーム連携重視
   - **教育/無料ツール**: 無料-低価格、基本機能のみ

2. **Video Noteの差別化ポイント**
   - Webベースでインストール不要
   - シンプルな共有リンク機能（認証不要）
   - 自動削除機能（7日保持）による運用負荷軽減
   - 500MBまでのアップロード（競合より大きい場合が多い）

3. **市場機会**
   - 月額1,000円（約$7-8/月）の中価格帯に位置決め可能
   - 日本語対応ツールが少ない
   - スポーツ分析ツールは高価格で個人利用のハードルが高い
   - 7日間無料トライアル+自動課金モデルで継続率向上

### 5.2 WebRTCとリアルタイム共同編集

調査によると、以下の技術がリアルタイム注釈に使用されている:

- **WebRTC**: P2Pビデオ通信用
- **Mesh/SFU/MCUアーキテクチャ**: マルチパーティ通信用
- **WebSocket**: リアルタイム同期

現在、主要な競合は**リアルタイム共同編集機能を提供していない**。これはVideo Noteの差別化機会となり得る。

### 5.3 Video Noteの技術的優位性

| 項目 | Video Note | 競合比較 |
|------|-----------|---------|
| **デプロイ** | Docker + Laravel Sail | 多くはSaaS/デスクトップ |
| **フロントエンド** | Vanilla JS（単一ファイル） | React/Vueが主流 |
| **バックエンド** | Laravel | Node.js/Pythonが主流 |
| **ストレージ** | シンプルファイルシステム | クラウドストレージ主流 |

## 6. Video Noteとの差別化ポイント

### 6.1 強み（競合優位性）

| 項目 | 詳細 |
|------|------|
| **1. Webベースで完全クラウド型** | インストール不要、どこからでもアクセス可能（Kinovea/Nacsportはデスクトップ） |
| **2. シンプルな共有機能** | 認証不要の共有リンク（Filestage等はアカウント要求） |
| **3. 自動削除による運用負荷軽減** | 7日保持、24時間前に通知（競合は無期限/手動削除が主流） |
| **4. 大容量アップロード** | 500MB（VideoAnt等はYouTubeのみ） |
| **5. 多様な注釈種類** | ペン/矢印/テキスト/番号/連番（VideoAntはテキストのみ） |
| **6. スナップショット機能** | タイムスタンプ付きスナップショット保存（Kinoveaと同等） |
| **7. 日本語対応** | 言語の壁がない（Hudl等は英語中心） |

### 6.2 弱み（改善機会）

| 項目 | 詳細 | 優先度 |
|------|------|--------|
| **1. リアルタイム共同編集なし** | 競合も実装していないが、将来的な差別化に | 中 |
| **2. モバイル非対応** | 2024年で必須機能になりつつある | 高 |
| **3. プラン差分未実装** | Proプランで上限緩和未実装 | 高 |
| **4. 単一巨大JSファイル** | 4,000行超で技術的負債 | 中 |
| **5. パブリックストレージ** | セキュリティ向上のため署名URL検討 | 中 |

### 6.3 ユニークな機能（競合にないもの）

| 機能 | 説明 |
|------|------|
| **連番自動付与** | 番号注釈で自動連番機能 |
| **自動削除通知** | 24時間前にメール通知 |
| **KPIイベント記録** | プロジェクト作成をDBに自動記録 |
| **readOnlyモード** | 共有時に編集不可モードを自動切替 |

### 7.2 推奨価格設定

**コスト構造の考慮事項:**
- サーバーコストは**プロジェクト数**に比例（ストレージ+DB）
- 動画ファイルの保存期間はコストに影響しない（ユーザーが削除すれば解放）
- フリーミアムは**7日間無料トライアル**とし、継続利用には自動課金

| プラン | 価格 | プロジェクト上限 | 試用期間 | 対象 |
|--------|------|-----------------|---------|------|
| **Free Trial** | ¥0 | 5 | 7日間 | 新規ユーザー全員 |
| **Basic** | ¥1,000/月 | 10 | - | 個人コーチ |
| **Standard** | ¥3,000/月 | 50 | - | チーム |
| **Pro** | ¥10,000/月 | 無制限 | - | 組織 |

**補足:**
- 7日間の無料トライアル期間終了後、自動的にBasicプラン（¥1,000/月）へ移行
- 途中解約可能（解約後は月末まで利用可能）
- プロジェクト数超過時は追加プラン購入か上位プランへ移行 |

| 優先度 | 機能 | 説明 |
